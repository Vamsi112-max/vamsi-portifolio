AI-Powered Enterprise Task Management

GenAI Task Management Assistant

Internal build PythonLLMPrompt EngineeringREST APISQL

An AI-powered enterprise task management platform to streamline company operations through intelligent task assignment, workflow automation, project tracking, approval management, and role-based access control.

Internal build · no public link

Problem

Teams needed a way to automate repetitive task assignment and approval workflows, and to interact with the system in plain language instead of navigating multiple manual steps.

My Role

I built this as part of my AI Developer Internship at Seventh Horse Agri-Tech (MATIC), working alongside the development team in Agile sprint reviews.

Key Features

  • A conversational AI assistant that understands natural-language requests
  • Automated workflows for task assignment and approvals
  • Project tracking and role-based access control
  • Real-time monitoring of employee and project data
  • On-demand report generation

Challenges

Getting the assistant to reliably interpret natural-language requests and map them to the correct workflow action required careful prompt engineering and testing.

What I Learned

This project sharpened my understanding of how to combine LLM-based assistants with REST APIs and SQL to automate real business workflows.
