Goal Intelligence & Performance Tracking SaaS Platform

AtomSphere Nexus

Live demo React.jsSupabaseSQLREST API

A multi-role SaaS platform to streamline employee performance management, goal tracking, project monitoring, KPI evaluation, review cycles, and organizational analytics through a centralized system.

Live demo →Source on GitHub

Problem

Organizations need a single place to track goals, KPIs, and performance reviews across different roles, instead of scattered spreadsheets and disconnected tools.

My Role

I built AtomSphere Nexus end to end — from database schema design to deployed, role-based dashboards.

Key Features

  • Role-based dashboards for Employees, Managers, and Admins
  • Real-time synchronization across the platform
  • Automated performance reviews and KPI evaluation
  • PDF/Excel report generation
  • Secure, centralized data management

Challenges

Designing role-based dashboards meant modeling permissions and data visibility carefully across Employee, Manager, and Admin roles while keeping real-time sync consistent across all of them.

What I Learned

Working with Supabase and React.js together taught me how to structure a multi-role SaaS data model and keep front-end state reliably in sync with a live backend.
